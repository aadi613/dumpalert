Key Features — Backed by Research
🔍
AI Waste Classification
Vision model identifies plastic, chemical, construction, organic, e-waste, or mixed waste from a single photo
⚖️
Weight Estimation
Human-mimetic approach — AI uses reference objects in the image (cars, bins, curbs) to estimate weight in kg
📍
Duplicate Detection
GPS clustering prevents the same dump being reported twice — warns user if a report exists within 100 m
⭐
Civic Credits
Gamified incentive system rewards citizens based on severity and waste weight to encourage reporting
